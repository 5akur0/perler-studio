> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

> **Executor note:** This plan is written for an agent with zero context on this repo. Read `CLAUDE.md`, `AGENTS.md`, and the spec `docs/superpowers/specs/2026-07-17-duo-mode-design.md` before starting. `DESIGN.md` governs all UI decisions. Feasibility was already verified live on 2026-07-17 (see spec).

**Goal:** Two players on separate devices place beads on the same pattern in real time, via a short-code room synced through the CloudBase realtime database.

**Architecture:** Room create/join goes through the existing `share-api` cloud function (short-code generation, member cap, rate limits). In-room sync bypasses the function: clients `watch()` the room document and write cell updates directly with `@cloudbase/js-sdk` (anonymous auth). Board sync is cell-level last-write-wins driven by a 400 ms local diff loop against a baseline snapshot — no changes to the existing bead-placement code paths in `main.js`.

**Tech Stack:** Vanilla ES modules + esbuild (committed bundles), CloudBase node-sdk (server) / js-sdk v2.17+ (client), plain Node regression scripts.

## Spec Amendments (decided at planning time)

1. **Cell keys are `i<flatIndex>`** (e.g. `i127`), not the spec's illustrative `r12_c07`. The board state `state.placed` is a flat array indexed by `row * width + col`; index keys avoid a row/col conversion layer and its off-by-one bug surface.
2. **Room codes are 8 chars** from the existing `SHORT_ID_ALPHABET` (`cloudbase/share-api/index.js:11`), not the spec's 6. This reuses the deployed generator, the client-side extraction regex, and keeps one short-code grammar product-wide.

```js
// src/duo-sync.js — pure sync core for duo mode. Board state travels as a flat
// "cells" object keyed by i<flatIndex>; values are MARD color codes. Removal is
// expressed as null in a diff and as key-absence in a snapshot. Cell-level
// last-write-wins: whoever's write lands later owns the cell.

const KEY_PATTERN = /^i(0|[1-9][0-9]*)$/;

export function cellKey(index) {
  return `i${index}`;
}

export function cellIndex(key) {
  const match = KEY_PATTERN.exec(String(key || ""));
  return match ? Number(match[1]) : -1;
}

export function snapshotCells(placed) {
  const cells = {};
  for (let i = 0; i < placed.length; i++) {
    if (placed[i]) cells[cellKey(i)] = placed[i];
  }
  return cells;
}

export function diffCells(baseline, placed) {
  const changes = {};
  for (let i = 0; i < placed.length; i++) {
    const key = cellKey(i);
    const local = placed[i] || null;
    const known = baseline[key] || null;
    if (local !== known) changes[key] = local;
  }
  // Baseline cells beyond placed.length (or cleared locally) count as removals.
  for (const key of Object.keys(baseline)) {
    const index = cellIndex(key);
    if (index >= placed.length && !(key in changes)) changes[key] = null;
  }
  return changes;
}

export function applyCells(placed, changes) {
  const changedIndexes = [];
  for (const [key, value] of Object.entries(changes || {})) {
    const index = cellIndex(key);
    if (index < 0) continue; // foreign/corrupt key: ignore, never throw mid-sync
    while (placed.length <= index) placed.push(null);
    const next = value || null;
    if ((placed[index] || null) === next) continue;
    placed[index] = next;
    changedIndexes.push(index);
  }
  return changedIndexes;
}

export function mergeBaseline(baseline, changes) {
  const next = { ...baseline };
  for (const [key, value] of Object.entries(changes || {})) {
    if (value) next[key] = value;
    else delete next[key];
  }
  return next;
}
```

```js
function normalizedRoomIdentity(payload) {
  const clientId = String(payload?.clientId || "").trim();
  const uid = String(payload?.uid || "").trim();
  const nickname = String(payload?.nickname || "").replace(/[\u0000-\u001F\u007F]/g, "").trim().slice(0, MAX_NICKNAME_LEN);
  if (!clientId || clientId.length > MAX_CLIENT_ID_LEN) {
    throw createHttpError(400, "bad_request", "Invalid clientId.");
  }
  if (!uid || uid.length > MAX_UID_LEN) {
    throw createHttpError(400, "bad_request", "Invalid uid.");
  }
  return { clientId, uid, nickname: nickname || "拼豆伙伴" };
}

async function createRoom(payload) {
  const identity = normalizedRoomIdentity(payload);
  const patternCode = String(payload?.patternCode || "");
  if (!patternCode || patternCode.length > MAX_PATTERN_CODE_LENGTH || !patternCode.startsWith("BEAM1")) {
    throw createHttpError(400, "bad_request", "Invalid pattern code.");
  }
  const name = String(payload?.name || DEFAULT_NAME).replace(/[\u0000-\u001F\u007F]/g, "").trim().slice(0, MAX_NAME_LENGTH) || DEFAULT_NAME;
  const now = Date.now();
  const member = { clientId: identity.clientId, uid: identity.uid, nickname: identity.nickname, joinedAt: now, lastSeen: now };
  for (let attempt = 0; attempt < MAX_SHORT_ID_ATTEMPTS; attempt++) {
    const code = randomShortId();
    try {
      await db.collection(ROOMS).add({
        _id: code,
        code,
        name,
        patternCode,
        stage: "placing",
        cells: {},
        members: [member],
        memberUids: [identity.uid],
        createdAt: now,
        expiresAt: now + ROOM_TTL_MS,
        updatedAt: now,
      });
      return { code, expiresAt: now + ROOM_TTL_MS };
    } catch (e) {
      // _id collision: retry with a fresh code; anything else propagates.
      if (attempt === MAX_SHORT_ID_ATTEMPTS - 1) throw e;
    }
  }
  throw createHttpError(500, "server_error", "Could not allocate a room code.");
}

async function joinRoom(payload) {
  const identity = normalizedRoomIdentity(payload);
  const code = String(payload?.code || "").trim();
  if (!new RegExp(`^[${SHORT_ID_ALPHABET}]{${SHORT_ID_LENGTH}}$`).test(code)) {
    throw createHttpError(400, "bad_request", "Invalid room code.");
  }
  return db.runTransaction(async (transaction) => {
    const docRef = transaction.collection(ROOMS).doc(code);
    const snap = await docRef.get();
    const room = firstDoc(snap);
    if (!room) throw createHttpError(404, "room_not_found", "Room does not exist.");
    if (Number(room.expiresAt) < Date.now()) throw createHttpError(410, "room_expired", "Room has expired.");
    const members = Array.isArray(room.members) ? room.members : [];
    let memberIndex = members.findIndex((m) => m && m.clientId === identity.clientId);
    const now = Date.now();
    if (memberIndex === -1) {
      if (members.length >= MAX_ROOM_MEMBERS) throw createHttpError(409, "room_full", "Room is full.");
      memberIndex = members.length;
      members.push({ clientId: identity.clientId, uid: identity.uid, nickname: identity.nickname, joinedAt: now, lastSeen: now });
    } else {
      // Rejoin: refresh uid (anonymous auth may have rotated) and presence.
      members[memberIndex] = { ...members[memberIndex], uid: identity.uid, nickname: identity.nickname, lastSeen: now };
    }
    const memberUids = members.map((m) => m.uid);
    await docRef.update({ members, memberUids, updatedAt: now });
    return {
      code,
      name: room.name,
      patternCode: room.patternCode,
      stage: room.stage,
      cells: room.cells || {},
      members,
      memberIndex,
      expiresAt: room.expiresAt,
    };
  });
}
```

```js
// src/duo-client.js — thin wrapper around @cloudbase/js-sdk for duo mode.
// Lazy-inits on first use so the SDK costs nothing for offline/solo players.
import cloudbase from "@cloudbase/js-sdk";

let app = null;
let db = null;
let signInPromise = null;

function envId() {
  return document.querySelector('meta[name="beam-cloudbase-env"]')?.content || "";
}

export function duoSupported() {
  return Boolean(envId())
    && window.location.protocol !== "file:"
    && navigator.onLine !== false;
}

function ensureApp() {
  if (!app) {
    app = cloudbase.init({ env: envId() });
    db = app.database();
  }
  return app;
}

export async function ensureSignedIn() {
  const auth = ensureApp().auth();
  if (!signInPromise) {
    signInPromise = (async () => {
      const state = await auth.getLoginState();
      if (!state) await auth.signInAnonymously();
      const user = auth.currentUser || (await auth.getLoginState())?.user;
      const uid = user?.uid;
      if (!uid) throw new Error("匿名登录失败");
      return uid;
    })().catch((e) => {
      signInPromise = null; // allow retry after a failure
      throw e;
    });
  }
  return signInPromise;
}

export function roomDoc(code) {
  ensureApp();
  return db.collection("rooms").doc(code);
}

export function dbCommand() {
  ensureApp();
  return db.command;
}
```

(If `auth.signInAnonymously` does not exist in the installed SDK version, fall back to `auth.anonymousAuthProvider().signIn()` — the 2026-07-17 experiment ran on 2.17.5 where `signInAnonymously` works.)

```js
  showToast: () => {},
  uiRenderUI: () => {},
};
export function setDuoActions(actions) {
  duoActions = { ...duoActions, ...actions };
}

const duo = {
  active: false,
  code: "",
  members: [],
  memberIndex: -1,
  baseline: {},
  watcher: null,
  syncTimer: null,
  heartbeatTimer: null,
  writing: false,
};

export function duoState() {
  return {
    active: duo.active,
    code: duo.code,
    members: duo.members,
    memberIndex: duo.memberIndex,
    peerOnline: peerOnline(duo.members, duo.memberIndex, Date.now()),
  };
}

export function peerOnline(members, memberIndex, now) {
  return (members || []).some(
    (member, index) => index !== memberIndex && member && now - Number(member.lastSeen) < PEER_ONLINE_WINDOW_MS,
  );
}

async function pushLocalChanges() {
  if (!duo.active || duo.writing) return;
  const changes = diffCells(duo.baseline, state.placed);
  const keys = Object.keys(changes);
  if (!keys.length) return;
  duo.writing = true;
  const optimistic = mergeBaseline(duo.baseline, changes);
  try {
    const _ = dbCommand();
    const payload = { updatedAt: Date.now() };
    for (const key of keys) {
      payload[`cells.${key}`] = changes[key] === null ? _.remove() : changes[key];
    }
    duo.baseline = optimistic; // optimistic: assume the write lands
    await roomDoc(duo.code).update(payload);
  } catch (e) {
    // Roll the failed keys back out of the baseline so the next tick retries them.
    for (const key of keys) {
      if (changes[key] === null) duo.baseline[key] = "?"; // force re-diff (placed has null, baseline non-null)
      else delete duo.baseline[key];
    }
    console.warn("duo push failed, will retry", e);
  } finally {
    duo.writing = false;
  }
}

function handleSnapshot(snap) {
  const doc = snap?.docs?.[0];
  if (!doc || !duo.active) return;
  duo.members = Array.isArray(doc.members) ? doc.members : duo.members;
  const cloudCells = doc.cells || {};
  // Remote changes = cloud vs baseline. Our own optimistic writes are already
  // in the baseline, so echoes diff to nothing.
  const remote = {};
  for (const [key, value] of Object.entries(cloudCells)) {
    if ((duo.baseline[key] || null) !== (value || null)) remote[key] = value || null;
  }
  for (const key of Object.keys(duo.baseline)) {
    if (!(key in cloudCells)) remote[key] = null;
  }
  // Cells the local player changed but hasn't pushed yet stay local (they win
  // the next push tick — deliberate LWW).
  const pendingLocal = diffCells(duo.baseline, state.placed);
  for (const key of Object.keys(pendingLocal)) delete remote[key];
  const changedIndexes = applyCells(state.placed, remote);
  duo.baseline = mergeBaseline(duo.baseline, remote);
  if (changedIndexes.length) {
    state.placedVersion += 1;
    state.renderDirty = true;
    state.uiDirty = true;
  }
  duoActions.uiRenderUI();
}

function startLoops() {
  stopLoops();
  duo.watcher = roomDoc(duo.code).watch({
    onChange: handleSnapshot,
    onError: (e) => {
      console.warn("duo watch error", e);
      duoActions.showToast("联机连接不稳定，正在重连…");
    },
  });
  duo.syncTimer = setInterval(pushLocalChanges, SYNC_INTERVAL_MS);
  duo.heartbeatTimer = setInterval(() => {
    if (!duo.active || duo.memberIndex < 0) return;
    roomDoc(duo.code)
      .update({ [`members.${duo.memberIndex}.lastSeen`]: Date.now() })
      .catch(() => {});
  }, HEARTBEAT_INTERVAL_MS);
}

function stopLoops() {
  try { duo.watcher?.close?.(); } catch { /* already closed */ }
  duo.watcher = null;
  if (duo.syncTimer) clearInterval(duo.syncTimer);
  if (duo.heartbeatTimer) clearInterval(duo.heartbeatTimer);
  duo.syncTimer = null;
  duo.heartbeatTimer = null;
}

async function enterRoom(joinData) {
  duo.code = joinData.code;
  duo.members = joinData.members || [];
  duo.memberIndex = joinData.memberIndex;
  duo.baseline = { ...(joinData.cells || {}) };
  duo.active = true;
  applyCells(state.placed, duo.baseline);
  state.placedVersion += 1;
  state.renderDirty = true;
  startLoops();
}

export async function duoCreateRoom() {
  if (!duoSupported()) throw new Error("当前环境不支持联机（需要联网访问）。");
  const uid = await ensureSignedIn();
  const pattern = state.selectedPattern;
  const data = await requestShareApi("/api/room/create", {
    patternCode: encodePatternCode(pattern),
    name: pattern?.name || "",
    clientId: getClientId(),
    uid,
    nickname: localStorage.getItem("beamDuoNickname") || "",
  });
  const joined = await requestShareApi("/api/room/join", {
    code: data.code,
    clientId: getClientId(),
    uid,
    nickname: localStorage.getItem("beamDuoNickname") || "",
  });
  await enterRoom(joined);
  return { code: data.code };
}

export async function duoJoinRoom(code) {
  if (!duoSupported()) throw new Error("当前环境不支持联机（需要联网访问）。");
  const uid = await ensureSignedIn();
  const joined = await requestShareApi("/api/room/join", {
    code: String(code || "").trim(),
    clientId: getClientId(),
    uid,
    nickname: localStorage.getItem("beamDuoNickname") || "",
  });
  const pattern = decodePatternCode(joined.patternCode, { name: joined.name || "联机图纸" });
  duoActions.loadPattern(pattern);
  await enterRoom(joined);
}

export function duoLeaveRoom() {
  stopLoops();
  duo.active = false;
  duo.code = "";
  duo.members = [];
  duo.memberIndex = -1;
  duo.baseline = {};
}
```

## Known Risks / Watch Items

- **SDK bundle weight**: accepted cost; if `app.bundle.js` grows past ~1 MB gzipped, flag it in the PR rather than silently shipping.
- **js-sdk auth API drift**: the plan targets 2.17.x (`signInAnonymously`). If npm installs a newer major, pin to `~2.17.5`.
- **Watch snapshot shape**: the experiment observed `snap.docs[0]` carrying the full doc on every event. If a future SDK delivers partial docs, `handleSnapshot` still behaves (it diffs whatever `cells` it sees against the baseline) but re-test the two-window flow.
- **Free-tier quotas**: after merging, watch `tcb env usage -e beam-prod-d7g2xz88ee6532631` for realtime connection/read spikes for a few days.
- **Do not** instrument bead-placement call sites in `main.js` for sync; the diff loop is the single integration point by design.
